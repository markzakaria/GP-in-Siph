# Silicon Photonic Transceiver Enabled by Mode division multiplexing

## Project Overview

This graduation project explores **silicon photonic-based Transceiver Enabled by Mode division multiplexing** for next-generation on-chip optical communication systems. The project combines photonic device modeling with optimization methods to improve performance, efficiency, and design exploration.

## My Role

My role focuses on the **simulation and optimization of silicon micro-ring modulators** for on-chip optical communication. This includes building and analyzing device models, evaluating optical and electrical performance, and supporting optimization workflows to improve modulator behavior.

## Objectives

* Model and simulate silicon photonic devices for optical interconnect applications.
* Study the behavior of silicon micro-ring modulators under different design and operating conditions.
* Apply optimization or learning-based methods to improve device performance.
* Evaluate key modulation metrics such as resonance tuning, Q factor, insertion loss, extinction ratio, and efficiency.
